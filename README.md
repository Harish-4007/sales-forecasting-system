# Sales Forecasting System

## Project Overview

This project predicts future retail sales using multiple forecasting techniques:

- ARIMA
- Prophet
- LSTM Neural Network

The goal is to compare forecasting performance and select the most accurate model.

---

## Dataset

Dataset: Walmart Weekly Sales Dataset

Source:
Kaggle

## Model Performance

| Model | MAE | RMSE |
|---------|---------|---------|
| ARIMA | 6910390 | 7567086 |
| Prophet | 6901725 | 8335014 |
| LSTM | 6510116 | 6849696 |

---

## Best Model

LSTM achieved the lowest RMSE and MAE values.

Selected as the final forecasting model.
